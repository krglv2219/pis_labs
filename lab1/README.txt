Лабораторная работа №1. Начало работы с Python.

Структура:
- helloworld.py — самая первая программа "Hello world".
- mygroup.py — список студентов и фильтрация по среднему баллу.

Шаги запуска (Windows):
1. Установите Python 3 с официального сайта python.org.
2. Откройте терминал (cmd или PowerShell).
3. Перейдите в папку lab1 командой:
   cd ПУТЬ_К_ПАПКЕ\lab1
4. Запустите:
   python helloworld.py
   python mygroup.py

Шаги запуска (macOS / Linux):
1. Установите Python 3 из менеджера пакетов или с python.org.
2. В терминале перейдите в папку lab1:
   cd /путь/к/папке/lab1
3. Запустите:
   python3 helloworld.py
   python3 mygroup.py
